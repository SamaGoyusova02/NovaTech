# NovaTech 

## Layihənin Qısa Təsviri
Bu layihə NovaTech şirkəti üçün hazırlanmış müasir, responsiv və əlçatan bir veb səhifədir. Sayt istifadəçilərə şirkətin xidmətləri, göstəriciləri, layihə qalereyası və interaktiv əlaqə formu vasitəsilə müraciət imkanı təqdim edir.

##  İstifade Olunan Texnologiyalar
* **HTML5:** Semantik teqlər və ARIA atributları ilə hazırlanmış struktur.
* **CSS3:** Mobile-first yanaşması, Flexbox və CSS Grid strukturları.
* **Vanilla JavaScript:** Mobil menyu interaktivliyi və Əlaqə Formunun RegExp ilə validasiyası.
* **Font Awesome:** İkonlar üçün.
